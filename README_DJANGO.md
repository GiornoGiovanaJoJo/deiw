# Django — Empire Premium

**Данные и все фото хранятся в SQLite** (один файл `db.sqlite3`): и текст, и изображения (в полях BinaryField). Папка `media/` не используется для загрузок.

## Установка и запуск

1. **Виртуальное окружение (рекомендуется)**
   ```bash
   python -m venv venv
   venv\Scripts\activate   # Windows
   ```

2. **Зависимости**
   ```bash
   pip install -r requirements.txt
   ```

3. **Миграции**
   ```bash
   python manage.py migrate
   ```

4. **Суперпользователь (для входа в админку)**
   ```bash
   python manage.py createsuperuser
   ```

5. **Запуск сервера**
   ```bash
   python manage.py runserver
   ```
   Сайт: http://127.0.0.1:8000/  
   Админка: http://127.0.0.1:8000/admin/

## Что настроено в админке

- **Настройки сайта** — логотип в шапке (загрузка изображения).
- **Карусель Hero** — до 10 изображений для блока справа в Hero. Если одно — показывается статично, если несколько — карусель (смена каждые 5 сек).
- **Услуги** — фото, заголовок, описание. Листаются стрелками (Следующая / Предыдущая).
- **Проекты** — фото, заголовок, описание, ссылка «Подробнее», адрес. По 3 на странице, под блоком — пагинация по номерам страниц.

## Структура

- `config/` — настройки проекта Django.
- `main/` — приложение: модели, представления, админка.
- `index.html` — шаблон главной (логотип, hero-карусель, услуги, проекты, пагинация).
- Изображения отдаются из БД по URL вида `/dbimg/<модель>/<id>/` и хранятся в SQLite (поля `*_data`, `*_type`, `*_name`).
